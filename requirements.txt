# --------------------------------------------------------------------
# Flujo real de la aplicación
# --------------------------------------------------------------------

# Alumno → HTTP POST /submit (FastAPI + Uvicorn)
#     │
#     ├─ Pydantic valida el payload (código Python del alumno)
#     ├─ JWT (python-jose) verifica que el alumno está autenticado
#     ├─ SQLAlchemy guarda el envío en PostgreSQL (vía psycopg2)
#     │
#     ├─ docker SDK arranca contenedor efímero (sandbox)
#     │     └─ ejecuta el código del alumno de forma aislada
#     │     └─ captura stdout/stderr
#     │     └─ mata el contenedor tras timeout o fin
#     │
#     ├─ Compara salida real vs. salida esperada
#     ├─ Calcula nota
#     └─ Devuelve JSON con nota + feedback al cliente

# Nota sobre versiones:
# El símbolo "==" significa versión exacta (ej. fastapi==0.111.0).
# Esto garantiza reproducibilidad por si luego hay en el futuro alguna
# actualización con cambios incompatibles, el entorno se crea
# exactamente igual en cualquier máquina, servidor o contenedor.
# El símbolo ">=" significa "esta versión o superior", usado solo
# cuando la compatibilidad está suficientemente garantizada.
# --------------------------------------------------------------------
