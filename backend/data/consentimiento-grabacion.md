# Consentimiento para supervision y evidencias

Durante la prueba se registraran eventos de auditoria del navegador, como cambio de pestana, perdida de foco o recuperacion de la ventana.

Antes de iniciar la prueba, el navegador solicitara permiso para compartir pantalla, audio de microfono y camara frontal. Estos permisos y la aceptacion del consentimiento son obligatorios para acceder al examen. Si se produce una incidencia, el navegador podra grabar una evidencia breve asociada al evento. La grabacion se usa solo para revision docente posterior y no aplica penalizacion automatica por si misma.

La validez del envio, el tiempo disponible y la calificacion oficial se deciden en el servidor. Si deniegas alguno de los permisos iniciales no se creara el intento. Si revocas un permiso durante una prueba ya iniciada, se conservara el evento de auditoria y quedara constancia de que no se pudo adjuntar evidencia.
