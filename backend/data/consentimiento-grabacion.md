# Consentimiento para supervision y evidencias

Durante la prueba se registraran eventos de auditoria del navegador, como cambio de pestana, perdida de foco o recuperacion de la ventana.

Si se produce una incidencia y has aceptado este consentimiento, el navegador podra solicitar permiso para grabar una evidencia breve asociada al evento: pantalla, audio de microfono y camara frontal. La grabacion se usa solo para revision docente posterior y no aplica penalizacion automatica por si misma.

La validez del envio, el tiempo disponible y la calificacion oficial se deciden en el servidor. Puedes denegar los permisos del navegador; en ese caso se conservara el evento de auditoria y quedara constancia de que no se pudo adjuntar evidencia.
